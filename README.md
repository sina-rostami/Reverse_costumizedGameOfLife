# Reversed Game Of Life

## Run

run **Main.cpp** and give input like this:

### Sample

input :

```
4 5 10
....*
..*.*
.*..*
..*.*
```

output :
```
..*.*
...*.
**.**
*...*
```

## Gtest

gtest tool used for test program. \
you can discover it in **test** directory.

### run gtest

you can run gtest by this commands :

```shell
cd test
mkdir build
cd build
cmake ..
make
./GolTest
```
